# 待办事项示例页面

这里提供了两个不同风格的待办事项示例页面，适合学习和使用。

## 文件说明

### 1. `todo.html` - 完整功能版
一个功能完整、界面美观的待办事项应用，包含以下特性：

**功能特性：**
- 添加新的待办事项
- 标记完成/未完成
- 删除待办事项
- 筛选功能（全部/未完成/已完成）
- 数据统计（总计/已完成/未完成）
- 本地存储（刷新页面数据不丢失）
- 响应式设计（适配手机和电脑）

**界面特点：**
- 现代化的渐变背景
- 平滑的动画效果
- 美观的卡片设计
- 直观的图标指示
- 良好的交互反馈

### 2. `todo-simple.html` - 初学者版
一个简洁明了的待办事项示例，适合初学者学习：

**功能特性：**
- 基本的添加功能
- 标记完成状态
- 删除功能
- 本地存储
- 简单明了的界面

**学习重点：**
- 基础的HTML结构
- 简单的CSS样式
- 基本的JavaScript操作
- DOM操作
- 本地存储API使用

## 使用方法

1. **直接打开**：双击HTML文件在浏览器中打开
2. **本地服务器**：可以使用以下方式运行：
   ```bash
   # Python 3
   python -m http.server 8000
   
   # 或使用Node.js的http-server
   npx http-server
   ```
